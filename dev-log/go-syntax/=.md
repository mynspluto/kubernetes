- =(지연 할당)
  다른 언어의 =와 같음 대입 연산

- :=(즉시 할당)
  const와 비슷 추후 변경 안됨

- ?=(조건부 할당)
  a ?= /usr/bin
  a가 정의되지 않았다면 값 대입

- +=(추가 할당)
  선언 되지 않은 변수도 사용 가능
  a += 'asd'하면 a에 asd 대입됨

- !=(쉘 할당)
  다른지 검사하는 비교 연산자와 똑같이 생겼으며 문맥에 따라 쉘 할당인지 비교 연산인지 구분됨
  쉘 명령어 실행하여 결과 변수에 대입
  밑의 두 코드 동작 동일함
  var1 := $(shell date)
  var2 != date
